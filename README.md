# Robotic Pest Control System 

 nano-robot system designed for precision agriculture applications, combining advanced pest detection capabilities with comprehensive soil health monitoring. The system leverages cutting-edge computer vision, IoT sensors, and machine learning algorithms to provide real-time agricultural insights. Smart Pest Detection is a project aimed at detecting pests using advanced machine learning algorithms. Garden Defender is a robotic garden pest deterrent. AI Quadruped for Agriculture is an OAK-D enabled quadruped robot with robotic arm to effectively spray water/pesticides/insecticides. The Autonomous Agricultural Robot utilizes AI, ROS, and advanced hardware to autonomously detect, pick, and transport paddy for efficient harvesting. MITRA is an AI-powered autonomous agricultural robot designed to solve the labor-intensive problem of manual weeding
